使用Gradle在Eclipse中构建Spring项目

在Windows下下载及安装Gradle

  1. Gradle下载地址

    http://www.gradle.org/downloads
  2. 在以上下载地址下载Gradle的binary安装包
  3. 解压下载的文件，并把解压后目录中的bin目录加到环境变量Path中

  4. 在cmd下测试Gradle是否安装成功，输入指令

    gradle

    如安装成功结果应该显示为

在Eclipse安装Gradle插件

  1. 官方的Eclipse Gradle插件叫buildship，具体可见

    https://github.com/eclipse/buildship/blob/master/docs/user/Installation.md

  2. 在eclipse中打开help->”Eclipse Marketplace”，在搜索栏中输入”buildship”

  1. 点击install,等待安装完成，按照提示重启eclipse即可
  2. 安装成功后，即可创建或导入Gradle项目

在Eclipse导入gradle项目

  1. 打开build.gradle文件，在第一行添加以下代码

    apply plugin 'eclipse'

  2. 在项目所在目录下运行命令

    gradlew.bat eclipse

  3. 在eclipse下导入该gradle项目
